# My Learning Journal

AI Agent × Web3 PM 求職備戰的學習知識庫。

主要在 claude.ai 學習並產出筆記，貼進 Claude Code 後負責歸檔、打卡、上傳 GitHub。

## 結構

```
├── nine-month-battle-plan.md      ← 求職備戰總綱（學習節奏的對應依據）
├── notes/               ← 各科學習筆記
│   ├── ai-agent/        ← AI Agent Manager 養成課程
│   ├── web3/            ← AI × Web3 School
│   ├── payment/         ← AI Agent Payment（暫停）
│   └── product/         ← Product Hunt 每週拆解
├── checkin/             ← 打卡：對應作戰書，依時間滾動彙整
│   ├── daily/           ← 一天一篇，多科分區
│   ├── weekly/
│   ├── monthly/         ← 對照作戰書月度 Milestone
│   ├── quarterly/
│   ├── half-year/
│   └── yearly/
├── _templates/          ← 打卡各層模板
└── profile.md           ← 使用者畫像與進度（agent 狀態）
```

## 打卡邏輯

貼筆記 → 歸檔到 `notes/<科>/` → 順手更新當天 `checkin/daily/` →
低層往高層滾動彙整（daily → weekly → monthly → quarterly → half-year → yearly），
每層都對應回 `nine-month-battle-plan.md` 的里程碑。
